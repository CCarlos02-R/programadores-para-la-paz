El comando uname -a muestra información del sistema operativo, como el nombre del sistema, la versión del kernel, la arquitectura del procesador y otros datos técnicos.
El comando whoami muestra el nombre del usuario que está utilizando la sesión actual.
Conocer esta información es útil para administrar un servidor porque permite identificar el sistema operativo, verificar el usuario que está ejecutando los comandos y facilitar tareas de configuración, mantenimiento y seguridad.
