Sistema operativo:
Es el software que administra los recursos del computador y permite ejecutar programas como Node.js.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador y crear servidores.

Aplicación server.js:
Es el archivo que contiene el código del servidor y define las rutas y respuestas de la aplicación.

Puerto:
Es el punto de comunicación por donde el servidor recibe las solicitudes de los usuarios. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que muestran la actividad del servidor, mensajes de funcionamiento y posibles errores para facilitar el monitoreo y la solución de problemas.
